# Video Chat App Build with WebRTC and Socket.io

[> Chat with friends online](https://rayc-video-chat.herokuapp.com/)

### Clone/Fork this repo
    git clone https://github.com/rayc2045/webrtc-video-chat-app.git
    npm i -g peerjs
    npm i
    npm run dev
    peerjs --port 3001
